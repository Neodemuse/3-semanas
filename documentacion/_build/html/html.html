Semántica¶

Etiqueta Uso
<header> Aqui van los titulos y logos, se puede poner uno por cada seccion
<section> Forma parte de algo mayor, su contenido trata un tema pero depende de otros
<article> Reutilizable, partes mas independientes
<div> Uso mas estetico y funcional que semantico aun asi el contenido tiene relacion cercana
<h1-h6> Para titulos, mientras menor el numero menos debe de haber en la pagina
<aside> Relacionado solo superficialmente con el contenido que lo rodea
<nav> Usado para bloques de enlaces a partes del sitio o la pagina
<footer> Copyright, autor, fecha, etc
<main> La informacion que el usuario busca, no barras, header o footer
<figure> | Se ponen las imagenes o diagramas junto con
una breve descripcion del mismo
